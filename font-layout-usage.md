**版面字体文件使用说明：**

- **通用字体：**
    - 标题缺失字符回退字体 ["FallbackFont"] : "SEGUISYM.TTF"

- **针对单曲的版面头版字体：**
    - 歌曲标题字体 ["TitleFont"] : "FOT-NewRodinProN-UB.otf"
    - 歌曲曲师字体 ["TitleFont"] : "FOT-NewRodinProN-UB.otf"
    - 歌曲ID/BPM/分类 ["MetaFont"] : "江城圆体 700W.ttf"

- **单曲成绩 ["maiMusicScore"]：**
    - 达成率/单曲Rat/DX分数字体 ["ScoretFont"] : "FOT-NewRodinProN-UB.otf"

- **单曲信息["maiMusicInfo"]:**
    - 各难度定数/Notes数量 ["ConstantFont"] : "RoGSanSrfStd-Bd.otf"
    - 谱师/版本/抽象画作者 ["AuthorFont"] : "江城圆体 700W.ttf"
